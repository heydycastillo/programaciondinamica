# Ejercicio Escaleras - Programacion Dinamica
# ESTUDIANTE : Heydy castillo 

## Descripcion

Este programa calcula la cantidad de formas posibles de subir una escalera de N escalones.

El personaje puede avanzar solamente:

- 1 escalon
- 2 escalones

El programa utiliza Programacion Dinamica para calcular las soluciones y muestra la tabla DP generada.


## Requisitos

Para ejecutar el programa se necesita:

- Tener instalado un compilador de C++.
- Tener un archivo con el codigo fuente del programa.
